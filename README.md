# MyIDE - 功能丰富的多语言IDE

MyIDE是一个基于PyQt6和QScintilla开发的功能丰富的多语言集成开发环境，支持多种编程语言，提供语法高亮、智能缩进、括号补全、实时语法检查等功能。

## 功能特性

### 📝 代码编辑功能
- **多语言支持**：支持Python、C++、Java、HTML、JavaScript、CSS、PHP、Bash、SQL、NASM等多种语言
- **语法高亮**：为每种语言提供语法高亮，使用QsciLexer实现
- **智能缩进**：自动缩进代码，支持括号缩进和控制流缩进
- **括号补全**：智能补全括号、引号等，避免在cout等流操作符情况下自动生成右括号
- **括号匹配**：高亮显示匹配的括号
- **自动保存**：每30秒自动保存所有打开的文件

### 🔍 语法检查
- **实时语法检查**：实时检查代码语法，在问题选项卡中显示错误和警告
- **精准的错误定位**：显示错误的行号、列号和详细描述
- **多语言支持**：为每种语言提供专门的语法检查
- **友好的错误提示**：提供易于理解的错误信息

### 💻 终端功能
- **集成PowerShell终端**：直接在IDE中使用PowerShell
- **多终端标签页**：支持创建多个终端标签页，最多5个
- **命令历史**：支持上下箭头导航命令历史
- **实时输出**：实时显示命令执行结果

### 📁 资源管理器
- **文件系统浏览**：浏览本地文件系统
- **双击打开文件**：支持双击文件在IDE中打开
- **目录展开/折叠**：支持展开和折叠目录
- **文件信息显示**：显示文件名、大小、类型和修改日期

### 🎨 界面设计
- **灵活的布局**：使用停靠窗口，支持自定义布局
- **标签页编辑**：支持多标签页编辑，可拖拽标签页
- **状态栏信息**：显示行号、列号、文件长度、行数、文件类型、编码等信息
- **语言选择**：支持从语言菜单选择当前文件的语言

### 📋 其他功能
- **自动打开changes.log**：首次运行自动打开更新日志
- **问题选项卡**：显示语法检查的错误和警告
- **视图切换**：支持显示/隐藏终端和资源管理器
- **跨平台支持**：支持Windows、Linux和macOS

## 安装方法

### 从源代码运行
1. 确保已安装Python 3.11或更高版本
2. 安装依赖：
   ```bash
   pip install PyQt6 PyQt6-Qsci
   ```
3. 运行IDE：
   ```bash
   python my_ide.py
   ```

### 从可执行文件运行
1. 下载编译好的可执行文件：`dist/my_ide.exe`
2. 双击运行即可

## 使用说明

### 基本操作
- **新建文件**：点击"文件"菜单 -> "新建"，或使用快捷键Ctrl+N
- **打开文件**：点击"文件"菜单 -> "打开"，或使用快捷键Ctrl+O
- **保存文件**：点击"文件"菜单 -> "保存"，或使用快捷键Ctrl+S
- **另存为**：点击"文件"菜单 -> "另存为"，或使用快捷键Ctrl+Shift+S
- **退出**：点击"文件"菜单 -> "退出"，或使用快捷键Ctrl+Q

### 语言选择
- 点击"语言"菜单，选择相应的语言
- 语言菜单采用层级结构，按字母顺序排列
- 支持自动检测文件类型并设置相应的语言

### 终端使用
- 点击"视图"菜单 -> "终端"，显示或隐藏终端
- 在终端中输入命令并按回车键执行
- 使用上下箭头导航命令历史
- 点击"+ 新建终端"按钮创建新的终端标签页

### 资源管理器
- 点击"视图"菜单 -> "资源管理器"，显示或隐藏资源管理器
- 双击文件在IDE中打开
- 双击目录展开或折叠

### 语法检查
- 自动实时检查语法，无需手动操作
- 问题选项卡显示所有错误和警告
- 点击问题列表中的项目，可跳转到相应的代码位置

## 功能详解

### 智能缩进
当在包含左大括号的行末按下回车键时，IDE会自动生成缩进的新行，例如：
```cpp
int main(){
    // 光标在这里，自动缩进
}
```

### 智能括号补全
- 自动补全括号、引号等
- 智能处理流操作符，避免在cout等情况下自动生成右括号
- 支持模板和函数调用的智能处理

### 实时语法检查
- 检查Python语法错误，如未闭合的字符串、缩进问题等
- 检查C++语法错误，如括号不匹配、缺少分号等
- 检查HTML标签嵌套和闭合
- 检查JavaScript语法错误
- 检查CSS属性格式

### 自动保存
- 每30秒自动保存所有打开的文件
- 确保在意外关闭IDE时不会丢失工作

## 技术栈

- **GUI框架**：PyQt6
- **文本编辑组件**：QScintilla (PyQt6-Qsci)
- **终端实现**：QProcess
- **文件系统操作**：QFileSystemModel
- **语法检查**：Python内置compile函数、正则表达式
- **编译工具**：PyInstaller
- **设置存储**：JSON

## 开发说明

### 项目结构
- `my_ide.py`：主IDE文件，包含所有功能实现
- `ide_settings.json`：IDE设置文件
- `changes.log`：更新日志
- `README.md`：项目说明文档

### 编译为可执行文件
使用PyInstaller编译为可执行文件：
```bash
pyinstaller --onefile --windowed --name my_ide my_ide.py
```

### 自定义设置
- IDE设置保存在`ide_settings.json`文件中
- 支持自定义自动保存间隔、是否自动打开changes.log等
